# guide-home-portal

## 项目简介

`guide-home-portal` 是一个用于归档和发布多个独立 HTML 页面的仓库。  
这里不针对任何特定域名或网站，仅作为静态页面的集合与分发入口，方便维护与访问。

## 目录结构

```
guide-home-portal/
├── index.html          # 入口页面，提供导航或说明
├── pages/              # 存放各独立 HTML 页面
│   ├── page1.html
│   ├── page2.html
│   └── ...
└── assets/             # 公共资源（CSS、JS、图片等）
    ├── css/
    ├── js/
    └── images/
```

- `index.html`：仓库的主页面，可列出所有归档页面的链接或简单说明。  
- `pages/`：所有独立 HTML 页面存放于此，按需命名。  
- `assets/`：共享资源文件，保持页面风格统一。

## 页面归档说明

- 每个 HTML 页面均为独立完整的文档，可直接通过浏览器打开。  
- 页面之间无强依赖关系，可根据需要单独更新或新增。  
- 所有页面均不包含任何具体的商业推广或诱导性内容，仅作为信息展示或功能演示。

## 维护说明

- 新增页面：在 `pages/` 下创建新 HTML 文件，并更新 `index.html` 中的导航链接。  
- 修改页面：直接编辑对应文件，建议保留版本历史以便回溯。  
- 删除页面：移除文件并清理 `index.html` 中相关链接。  
- 资源更新：如需更换公共样式或脚本，请在 `assets/` 下统一修改。

## 使用方式

1. 克隆或下载本仓库。  
2. 直接打开 `index.html` 查看页面导航。  
3. 点击对应链接跳转到具体页面，或直接打开 `pages/` 下的文件。
